JavaScript es un lenguaje interpretado, esto quiere decir que se lee y se traduce a lenguaje maquina a medida que el programa se va ejecutando linea a linea

Por otro lado JavaScript es un lenguaje que se ejecuta esencialmente en un navegador web, como lo es google chrome en windows o safari en mac por ejemplo.

Java por otro lado es un lenguaje compilado esto quiere decir que el codigo es traducido primero a lenguaje maquina para luego ser ejecutado. Se dice que es mas veloz

Cabe destacar que el codigo en JavaScript se ejecuta del lado del cliente mientras que Java se ejecuta desde un equipo local o desde el servidor.
